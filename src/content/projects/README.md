# Projects content

Use `src/data/projects.ts` for the public project index and dynamic project pages.

When a project needs a longer editorial case study, add an MDX file here with the same slug:

```txt
src/content/projects/silph-technologies.mdx
```

Suggested frontmatter:

```yaml
title: "Project title"
slug: "project-slug"
status: "published"
date: "YYYY-MM-DD"
```
